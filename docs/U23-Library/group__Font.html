Draws Text to a Bitmap.

Parameters
destThe Bitmap the text will be drawn on
textString you want to draw
xx-coordinate for the text
yy-coordinate for the text

Before calling DrawText you need to call setFont once to set a font. Draw Text will use this font until you set a different one. Newlines are supported, text will continue at x / y + width of font Usage example: DrawText(surface, "Bla\nBlub", 10, 10);

void setFont ( const RLEBitmap *const *  font)

Sets the font that is used by DrawText.

Parameters
fontPointer to the font that will be used by DrawText

This needs to be called before you can use DrawText. Usage example: setFont(fontblack8);
