Detailed Description

Interface to the USART.

USART stands for "universal synchronous/asynchronous receiver/transmitter" This is used to debug your code via a serial console. Do not use this directly, use printf() instead. On your Computer do the following to communicate with your STM32 

# start screen. the device may vary. ask or google for help.
$ screen /dev/ttyUSB0 115200

# to end the screen session:
^A
